French Language Translations for Monstroid.

== Description ==

Traductions françaises du Thème Monstroid, vous permet de l'utiliser en toute liberté et de pouvoir bénéficier de la mise à jour des traductions.

== Installation ==

1. Télécharger le fichier zip de l'extension
2. Installer l'extension en cliquant sur via "Extensions" -> "Ajouter".
3. Activer votre licence dans le menu "Extensions" -> "Licence Monstroid French".
4. C'est tout, votre thème est traduit !
