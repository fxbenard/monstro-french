French Language Translations for Divi.

== Description ==

Traductions françaises du Thème Divi d'Elegant Thèmes, vous permet d'utiliser Divi-Child en toute liberté et de pouvoir bénéficier de la mise à jour des traductions.

**Related Links:**

* https://www.fxbenard.com/

== Screenshots ==


== Installation ==

1. Télécharger le fichier zip de l'extension
2. Installer l'extension en cliquant sur via "Extensions" -> "Ajouter".
3. Activer votre licence dans le menu "Extensions" -> "Licence Divi French".
4. C'est tout, votre thème est traduit !
